# Wally

Simple and Responsive Wallpaper App using Pexels API.

![banner](https://user-images.githubusercontent.com/96806599/214118500-03485ef4-c631-49de-b520-9321625797cc.png)


## Installation

Clone the repo

```bash
  flutter pub get
```
```bash
  flutter run
```
